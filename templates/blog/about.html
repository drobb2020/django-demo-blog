About Excession Development and this Blog

Coded By:

David Robb (Sunday, May 1, 2022)

Welcome to an updated version of the Python Django blog project based on the YouTube tutorial series hosted by John Elder of Codemy.com here.

The original project was based on an earlier version of Python, django, and bootstrap. This project has been updated to the latest version of Django 4 and Bootstrap 5.

This project will not be hosted publicly, but on a private server in my lab environment, and will primarily be used to host technical posts, except those from Abby.
